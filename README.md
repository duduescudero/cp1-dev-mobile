## Descrição do Projeto

Este projeto foi desenvolvido como parte da atividade **CP1** da disciplina **Desenvolvimento Mobile**.

A proposta do trabalho consiste na criação de uma **tela de login inspirada no aplicativo da FIAP**, utilizando **React Native** com **Expo**. O objetivo principal é aplicar conceitos fundamentais de desenvolvimento mobile, incluindo a construção de interfaces, organização de componentes, estilização visual e interação com elementos da tela.

A aplicação foi pensada para representar uma tela moderna, limpa e funcional, simulando uma experiência de autenticação de usuário em um aplicativo institucional.

---

## Objetivo da Atividade

O principal objetivo deste projeto é exercitar a criação de interfaces mobile utilizando os conceitos básicos e intermediários vistos em aula, como:

- Estruturação de projeto em pastas
- Criação de componentes reutilizáveis
- Uso de propriedades e estados
- Manipulação de inputs
- Interação com botões e switch
- Estilização de componentes com `StyleSheet`
- Organização visual inspirada em um app real

---

## Funcionalidades Implementadas
